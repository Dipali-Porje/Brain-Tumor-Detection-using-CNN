# Brain Tumor Detection using CNN with Django Dashboard

This project is a deep learning-based web application that detects brain tumors from MRI images using a Convolutional Neural Network (CNN). The backend is built with Django, and the model is integrated into an interactive dashboard for real-time predictions.

## Features

- Upload brain MRI images for prediction
- Real-time tumor detection using CNN
- Django-powered web dashboard
- Color-coded results for easy interpretation
- Organized and user-friendly UI

---

## Tech Stack

- **Frontend**: HTML, CSS, Bootstrap
- **Backend**: Django (Python)
- **Model**: CNN using TensorFlow / Keras
- **Other Tools**: OpenCV, NumPy, Pillow

---

## How It Works

1. User uploads an MRI image via the dashboard.
2. Image is preprocessed and passed to the trained CNN model.
3. Model predicts whether the brain scan has a tumor or not.
4. Result is displayed with visual feedback.
